# Theme & Colors

All color decisions in the platform flow from two sources: the **MUI theme** defined in `@ot/config/src/theme.ts` and the **data-visualization palettes** defined in `@ot/constants` and the AssociationsToolkit.

---

## Theme palette

The two brand colors that seed the MUI theme. `primaryColor` and `secondaryColor` are set in the runtime config profile and can be overridden per deployment.

## Data visualisation

A 9-stop blue ramp (`ASSOCIATION_COLORS` in `associationsUtils/index.ts`) mapped with `d3.scaleQuantize` over the `[0, 1]` domain. Low scores get the pale end; high scores get the saturated end.
